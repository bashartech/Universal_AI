Universal Chatbot - Embed Instructions

Step 1: Copy the Embed Code

Add this code snippet before the closing </body> tag of your website:

<!-- Universal Chatbot Widget -->
<script src="https://your-domain.com/chatbot.js"></script>
<script>
  // Optional: Initialize with custom configuration
  UniversalChatbot.init({
    widgetId: 'your-business-id',
    position: 'bottom-right', // or 'bottom-left'
    theme: {
      primaryColor: '#3B82F6'
    }
  });
</script>
Note: Replace https://your-domain.com/chatbot.js with the actual URL where you host the chatbot script.

Step 2: Customize Your Configuration

Edit the businessConfig.json file to match your business details:

{
  "business_name": "Your Business Name",
  "industry": "Your Industry",
  "services": ["Service 1", "Service 2", "Service 3"],
  "pricing": "Your pricing information",
  "business_hours": "Your business hours",
  "contact_email": "your@email.com",
  "contact_phone": "+XX-XXX-XXXXXXX",
  "quick_replies": ["Option 1", "Option 2", "Option 3"]
}

Step 3: Test the Widget

The chatbot widget should now appear on your website. Click the chat button in the bottom right corner to test it!

Platform-Specific Instructions

WordPress

1. Go to Appearance → Theme Editor
2. Edit footer.php
3. Paste the embed code before </body>

Wix

1. Go to Settings → Custom Code
2. Click "+ Add Custom Code"
3. Paste the embed code
4. Set to load on "All Pages" and "Body - end"

Shopify

1. Go to Online Store → Themes
2. Click "Actions" → "Edit code"
3. Edit theme.liquid
4. Paste the embed code before </body>

Webflow

1. Go to Project Settings → Custom Code
2. Paste the embed code in "Footer Code"
3. Publish your site
